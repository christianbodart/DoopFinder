   - Click any file to open a preview.

   - Usually, keep one copy and remove the rest.
   - DoopFinder marks the file that will be kept for you.

6. **Delete Selected Files**
   - Click the "Delete Selected" button.
   - The program moves duplicates to the Recycle Bin or Trash, so you can restore if needed.
